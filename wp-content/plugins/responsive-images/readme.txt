Choose the size of image show in your post when the readers access from mobile devices.

== Description ==

Choose the size of image show in your post when the readers access from mobile devices. This plugin will detect the device of the reader and load the size of the imgae that you have been selected previously at its control panel.


== Installation ==

1. Upload `social-fellow` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
